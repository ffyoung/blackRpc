①blackRpc是一个简单的分布式RPC框架                                                                                                           
②   框架组成：spring,netty,zookeeper                                                                                                       
    序列化方式支持：fastjson,msgpack,protostuff                                                                                             
    集群负载均衡策略：轮训，加权轮训，随机，加权随机，一致性哈希                                                                                 
③该框架比较适合初探分布式RPC原理的读者。                                                                                                      
④支持spring多种作用域
